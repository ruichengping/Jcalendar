Jcalendar

    new Jcalendar({
        input:"demo1",
    });
            
    new Jcalendar({
        input: "demo2",
        timeSelector: true,
    });
            
    new Jcalendar({
        input: "demo3",
        timeSelector: true,
        title: "请选择开始时间"
    });

参数

属性 取值类型 取值范围 说明
input String 字符串 input框的id值
timeSelector Boolean true或者false 是否带时间选择，默认是false
title String 字符串 时间选择器标题，默认是”请选择“
